## ✨ 首次发布 v1.0.0

🐯 虎哥API反代 - Cloudflare 反向代理 IP 优选工具

### 功能特点

- 🚀 **IP 测速** - 自动测试多个 Cloudflare IP 的 TCP 延迟，找出最快节点
- 📝 **Hosts 管理** - 一键修改系统 hosts 文件，无需手动编辑
- 🔍 **Hosts 查看器** - 简洁界面，查看和管理所有 hosts 配置
- 💾 **配置持久化** - 自动保存配置，下次启动自动加载
- 🔐 **管理员权限** - 自动请求管理员权限，安全修改系统文件

### 📦 下载

下载 `虎哥API反代.exe` 即可使用。

### 使用说明

1. 下载 EXE 文件
2. 双击运行，允许 UAC 管理员权限请求
3. 配置目标节点和 CF 反代域名
4. 点击「开始测速」找到最快 IP
5. 点击「应用最佳 IP」写入 hosts
